Welcome to the Appium Safari Driver documentation! The Safari driver is a test automation framework
for the Safari browser on macOS, iOS and iPadOS, enabling automated black-box testing on both
emulators and real devices. 

The Safari driver is part of the Appium test automation tool. For information on Appium itself,
please visit [the Appium documentation](https://appium.io).
